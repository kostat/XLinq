# XLinq
- ExTree Converts MSIL to Expression tree, so a compiled method can be converted to `Expression`. Call `ExpressionTree.Parse(...)`. [![nuget](https://img.shields.io/nuget/vpre/Streamx.Linq.ExTree?label=ExTree)](https://www.nuget.org/packages/Streamx.Linq.ExTree)
- SQL.Net.EFCore implements Linq to SQL for EF Core. It allows you to use C# (or your .NET language of choice) to write strongly typed queries. Interactive tutorial available at https://try.entitylinq.com/
